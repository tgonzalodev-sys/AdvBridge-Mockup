# Task 06.2 — Monthly detail drawer UI

## Objective
Build the detailed monthly control drawer.

## Scope
- show lines by currency
- fill variable amount
- edit pending lines
- add/remove manual billables
- statuses visible
- warnings for incomplete lines

## Out of scope
- full-month invoiced block rule
- advanced history

## Acceptance criteria
- Users can complete the monthly workflow inside the drawer
- Pending/manual actions behave correctly

## Stop and ask if
- line editing needs inline row editing vs form modal decision
