# Task 01.4 — Create/edit client flows

## Objective
Implement create and edit flows for Madre and Hija.

## Scope
- Create Madre form
- Edit Madre form
- Create Hija from Madre detail
- Edit Hija form
- Hija country optional
- subsidiary inherited and hidden on Hija
- dangerous deletes use confirmation

## Out of scope
- create Hija from contract flow
- history visualization

## Acceptance criteria
- Users can create and edit Madre/Hija
- subsidiary cannot be changed on Madre after creation
- validation errors are clear

## Stop and ask if
- Delete/archive decisions need to be encoded now in UI
