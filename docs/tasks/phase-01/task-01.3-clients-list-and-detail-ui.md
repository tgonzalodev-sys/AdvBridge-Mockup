# Task 01.3 — Clients list and detail UI

## Objective
Build the main clients UI.

## Scope
- Clients table
- Filters:
  - subsidiary
  - country
  - group
  - status
- Search
- One row per Madre by default
- Optional inline expansion for Hijas
- Madre detail page
- Show key fields and Hija list

## Out of scope
- contract summary box beyond small contextual data
- contacts
- dashboard

## Acceptance criteria
- Client list usable end to end
- Users can expand Hijas inline
- Madre detail opens and shows current values

## Stop and ask if
- Table complexity requires a dependency not yet approved
