# Task 00.1 — Repo bootstrap

## Objective
Create the base Next.js application and the repo structure that future tasks will build on.

## Scope
- Initialize Next.js App Router project
- Configure pnpm usage
- Create baseline folder structure:
  - src/app
  - src/components
  - src/modules
  - prisma
  - tests
  - docs
- Add placeholder root layout and homepage
- Keep the app empty but running

## Out of scope
- Prisma schema
- Domain features
- Styling polish
- Authentication

## Acceptance criteria
- Project installs and runs
- Folder structure exists
- App boots without errors
- Docs are present in repo

## Required checks
- pnpm install
- pnpm build

## Stop and ask if
- Bootstrap choice conflicts with App Router or TypeScript strict setup
